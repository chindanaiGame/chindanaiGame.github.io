หลักการของ CSS Specificity

CSS Specificity คือ ค่าที่จะจะบอก browser ว่าให้เลือกใช้ Style ตัวไหนมาแสดงผลหน้าเว็บ และจะต้องมีการคำนวนค่าออกมาเพื่อให้ browser รู้ เช่น

p {
color: yellow;
}

โดยการคำนวณค่า CSS Specificity นั้นจะนิยมใช้ selector การเขียนโดยใช้ selector ก็จะมีดังนี้

- Type selectors คือการใช้ element เป็น selector เช่น h1
- Class selectors เช่น .container
- ID selectors คือการใช้ id เป็น selector เช่น #name, #profile

ในส่วนของการคำนวณนั้นสิ่งต้องห้ามอย่างแรกเลยคือ !important

!important คือ คำสั่งในการกำหนดลำดับความสำคัญว่า Selector นี้สำคัญที่สุด ที่ไม่ควรใช้ก็เพราะว่าเวลาเราเขียนโค้ดเยอะแล้วจะให้มีปัญหาตีกับ Element อื่นๆตลอด
